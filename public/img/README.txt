Optional: add an equirectangular Earth texture here for the 3D globe.

- Filename: earth.jpg (or earth.webp)
- Format: equirectangular (2:1 aspect, e.g. 2048x1024 or 4096x2048)
- If present, the app loads /img/earth.jpg for the globe map.
- If not present, the app falls back to a CDN texture.

Free sources: NASA Blue Marble, Natural Earth, etc.
